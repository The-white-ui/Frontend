Cards

Available classes in cards:

  • .card: to create a new card.
  • .card-header: to create a card header inside a card.
  • .card-body: to create a wrapper for the owe card content.
  • .card-image: to create a wrapper or a container for an img inside a card.
  • .card-footer: to create a card footer inside a card.
