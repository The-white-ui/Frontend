Forms

Examples and usage guidelines for form control styles, layout options, and custom components for creating a wide variety of forms.

Forms Examples

Form Codes

                        
                            <div class="card">
                                <div class="form-group">
                                  <input type="..." class="input-field" placeholder="..."/>
                                  <input type="..." class="input-field" placeholder="..."/>
                                  <div class="button-container">
                                      <button class="btn-primary">
                                        Submit
                                      </button>
                                    </div>
                                </div>
                            </div>

Available classes in cards:

  • .form-group: to create a wrapper for the input fields.
  • .input-field: contain padding and margin for input tags.
  • .button-container: to create a wrapper for button.
