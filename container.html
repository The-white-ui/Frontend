Containers

Containers are a fundamental building block of White-uicss that contain, pad, and align your content within a given device or viewport.

How they Work

Containers are the most basic layout element in White-uicss and are required when using our default grid system. Containers are used to contain, pad, and (sometimes) center the content within them. While containers can be nested, most layouts do not require a nested container.

We have two different containers:

  • .container, which set max-width at each responsive breakpoint.
  • .container-full, which is 100% width of all breakpoints.

Default container

Our default .container class is a responsive, fixed-width container, meaning its max-width changes at each breakpoint.

                  
                      <div class="container">
                        <!--Content here-->
                      </div>

Container Full

Use .container-full for a full width container, spanning the entire width of the viewport.

                  
                      <div class="container">
                        <!--Content here-->
                      </div>
