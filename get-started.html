Introduction

Learn how to easily start using White-UI in your website.

Quick start

Looking to quickly add White-UI to your project? Use jsDelivr, a free open source CDN. To download the source files? Click Here to downloads.

CSS

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS. 

                        
                            <
                            link
                            href
                            =
                            "http://example.com"
                            />

JS

Many of our components require the use of JavaScript to perform their functions. Specifically, they require our own JavaScript plugins. Place the below code <script> near the end of your pages, right before the closing </body> tag, to enable them. 

                        
                            <
                            link
                            href
                            =
                            "http://example.com"
                            />

HTML Setup

Next you just have to make sure you link the files properly in your webpage. Generally it is wise to import javascript files at the end of the body to reduce page load time. Follow the example below on how to import Materialize into your webpage. 


                    <!DOCTYPE html>
                    <html>
                      <head>
                        <!--Import White-UIFW.css-->
                        <link type="text/css" rel="stylesheet" href="css/White-UIFW.min.css"  media="screen,projection"/>
                  
                        <!--Let browser know website is optimized for mobile-->
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
                      </head>
                  
                      <body>
                  
                        <!--JavaScript at end of body for optimized loading-->
                        <script type="text/javascript" src="js/White-UIFW.min.js"></script>
                      </body>
                    </html>
