Buttons

Use White-UI custom button styles in forms, and more with support for multiple sizes, background color, and more.

Example Buttons

Default Primary Secondary Error info Purple Orange Yellow 
                             
                             <button type="button" class="btn ">default</button>
                             <button type="button" class="btn btn-primary">Click me</button>
                             <button type="button" class="btn btn-secondary">Click me</button>

                             <button type="button" class="btn btn-info">Click me</button>
                             <button type="button" class="btn btn-outlined-purple">Click me</button>

Button tags

The .btn classes are designed to be used with the <button> element. However, you can also use these classes on <a> or <input/> elements (though some browsers may apply a slightly different rendering). When using button classes on <a> elements that are used to trigger in-page functionality (like collapsing content), rather than linking to new pages or sections within the current page, these links should be given a role="button" to appropriately convey their purpose to assistive technologies such as screen readers.

Link 
                             
                             <a href="#" class="btn-secondary text-white ">Link</a>
                             <button type="submit" class="btn-secondary text-white ">Button</button>
                             <input type="button" class="btn-secondary text-white " value="Input"/>
                             <input type="submit" class="btn-secondary text-white " value="Submit"/>
                             <input type="reset" class="btn-secondary text-white " value="Reset"/>

Outline buttons

In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the .btn-outline-* classes are designed to be used with the <button> element. 

                            
                             <button type="submit" class="btn-outlined-secondary text-secondary text-hover-white ">Secondary</button>
                             <button type="submit" class="btn-outlined-primary text-primary text-hover-white ">Primary</button>
                             <button type="submit" class="btn-outlined-purple text-purple text-hover-white ">Purple</button>
                             <button type="submit" class="btn-outlined-yellow text-yellow text-hover-white ">Yellow</button>
                             <button type="submit" class="btn-outlined-black text-balck text-hover-white ">Balck</button>
                             <button type="submit" class="btn-outlined-info text-info text-hover-white ">Info</button>
                             <button type="submit" class="btn-outlined-red text-red text-hover-white ">Red</button>

Complement buttons

In need of a button to complement your project, Replace the default modifier classes with the .btn-complement* classes are designed to be used with the <button> element. 

                            
                             <button type="submit" class="btn-complement-secondary ">Secondary</button>
                             <button type="submit" class="btn-complement-primary ">Primary</button>
                             <button type="submit" class="btn-complement-purple ">Purple</button>
                             <button type="submit" class="btn-complement-yellow ">Yellow</button>
                             <button type="submit" class="btn-complement-black ">Black</button>
                             <button type="submit" class="btn-complement-info ">Info</button>
                             <button type="submit" class="btn-complement-red ">Red</button>
